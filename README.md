# IntelliSpark AI – Electrical Assistant

## Project Overview
IntelliSpark AI is an Electrical Assistant Chatbot developed to help users learn basic electrical and electronics concepts through an interactive chat interface.

## Features
- User Login System
- AI Chat Interface
- Electrical Question & Answer Support
- Fast Response
- Simple User Interface
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Flask
